The `heading` node is used to represent blocks of heading in the document.

<Demo story="heading" />

## Usage

```ts twoslash
import { defineHeading } from 'prosekit/extensions/heading'

const extension = defineHeading()
```

## Commands

### `setHeading`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHeading } from 'prosekit/extensions/heading'

const extension = defineHeading()
const editor = createEditor({ extension })
// ---cut---
editor.commands.setHeading({ level: 1 })
```

### `toggleHeading`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHeading } from 'prosekit/extensions/heading'

const extension = defineHeading()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleHeading({ level: 1 })
```

### `insertHeading`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHeading } from 'prosekit/extensions/heading'

const extension = defineHeading()
const editor = createEditor({ extension })
// ---cut---
editor.commands.insertHeading({ level: 1 })
```

## Keyboard Shortcuts

| Non-Apple                     | Apple                               | Description                                                                 |
| ----------------------------- | ----------------------------------- | --------------------------------------------------------------------------- |
| <Keyboard key="Ctrl+Alt+1" /> | <Keyboard key="Command+Option+1" /> | Set the current block to an H1 node                                         |
| <Keyboard key="Ctrl+Alt+2" /> | <Keyboard key="Command+Option+2" /> | Set the current block to an H2 node                                         |
| <Keyboard key="Ctrl+Alt+3" /> | <Keyboard key="Command+Option+3" /> | Set the current block to an H3 node                                         |
| <Keyboard key="Ctrl+Alt+4" /> | <Keyboard key="Command+Option+4" /> | Set the current block to an H4 node                                         |
| <Keyboard key="Ctrl+Alt+5" /> | <Keyboard key="Command+Option+5" /> | Set the current block to an H5 node                                         |
| <Keyboard key="Ctrl+Alt+6" /> | <Keyboard key="Command+Option+6" /> | Set the current block to an H6 node                                         |
| <Keyboard key="Backspace" />  | <Keyboard key="Backspace" />        | When the text cursor is at the start of a heading node, set it to paragraph |

## Keyboard Interaction

Type `#` and a space to create a `heading`. One `#` makes an H1 heading, two `##` make an H2 heading, and so on.
