# @prosekit/ai

AI-related extensions for ProseKit.

## Installation

```bash
pnpm add @prosekit/ai
```

## Usage

```typescript
import { defineAIExtension } from '@prosekit/ai'

const extension = defineAIExtension()
```

## Development

To build this package:

```bash
pnpm run build
```

To run tests:

```bash
pnpm run test
```
