Add a new option `presence` to `defineLoro()`, which is a `CursorEphemeralStore` instance. `CursorEphemeralStore` is a better alternative to the legacy `Awareness`. If you're using `loro-prosemirror`, the minimal required version of `loro-prosemirror` is now `0.4.0`.
