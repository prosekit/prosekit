The `horizontalRule` node is used to represent a thematic break `<hr>` in the document, typically rendered as a horizontal line.

## Usage

```ts twoslash
import { defineHorizontalRule } from 'prosekit/extensions/horizontal-rule'

const extension = defineHorizontalRule()
```

## Commands

### insertHorizontalRule

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHorizontalRule } from 'prosekit/extensions/horizontal-rule'

const extension = defineHorizontalRule()
const editor = createEditor({ extension })
// ---cut---
editor.commands.insertHorizontalRule()
```

## Keyboard Interaction

Type three hyphens (`---`) on an empty line. This will automatically create a horizontal rule.
