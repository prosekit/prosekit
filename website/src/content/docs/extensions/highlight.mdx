The `highlight` mark is used to represent highlighted text, like using a highlighter pen. It will be rendered as `<mark>` element in HTML.

<Demo story="highlight" />

## Commands

### `toggleHighlight`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHighlight } from 'prosekit/extensions/highlight'

const extension = defineHighlight()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleHighlight()
```

## Keyboard Shortcuts

| Non-Apple                       | Apple                              | Description                               |
| ------------------------------- | ---------------------------------- | ----------------------------------------- |
| <Keyboard key="Ctrl+Shift+H" /> | <Keyboard key="Command+Shift+H" /> | Toggle the current selection to highlight |
| <Keyboard key="Ctrl+Alt+H" />   | <Keyboard key="Command+Alt+H" />   | Toggle the current selection to highlight |

## Input Rules

Type `==text==` to automatically apply the highlight mark to "text".
