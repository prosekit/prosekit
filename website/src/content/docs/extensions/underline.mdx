The `underline` mark is used to represent text with a underline. It will be rendered as `<u>` element in HTML.

<Demo story="underline" />

## Commands

### `toggleUnderline`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineUnderline } from 'prosekit/extensions/underline'

const extension = defineUnderline()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleUnderline()
```

## Keyboard Shortcuts

| Non-Apple                 | Apple                        | Description                               |
| ------------------------- | ---------------------------- | ----------------------------------------- |
| <Keyboard key="Ctrl+U" /> | <Keyboard key="Command+U" /> | Toggle the current selection to underline |
