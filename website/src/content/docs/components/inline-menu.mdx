An inline menu is a floating panel that appears over the current selection and disappears when the selection collapses. Use it for selection-relative actions: format the highlighted text, link it, or wrap it in a node.

## Structure

```
InlinePopoverRoot                # tracks the current selection and toggles the popover
└── InlinePopoverPositioner      # anchors the popup to the selection rect
    └── InlinePopoverPopup       # the floating button bar
        └── Button               # actions like "toggle bold", "open link", etc.
```

The primitive itself only handles positioning and visibility. You supply the buttons and form. Use `useEditorDerivedValue` to subscribe to `isActive` / `canExec` for each command, exactly like a [toolbar](/components/toolbar).
