The `code` mark is used to represent a short fragment of computer code. It will be rendered as `<code>` element in HTML.

<Demo story="code" />

## Commands

### `toggleCode`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineCode } from 'prosekit/extensions/code'

const extension = defineCode()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleCode()
```

## Keyboard Shortcuts

| Non-Apple                 | Apple                        | Description                          |
| ------------------------- | ---------------------------- | ------------------------------------ |
| <Keyboard key="Ctrl+E" /> | <Keyboard key="Command+E" /> | Toggle the current selection to code |
