The `hardBreak` node is used to represent a line break `<br>` in the document.

<Demo story="hard-break" />

## Usage

```ts twoslash
import { defineHardBreak } from 'prosekit/extensions/hard-break'

const extension = defineHardBreak()
```

## Commands

### `insertHardBreak`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineHardBreak } from 'prosekit/extensions/hard-break'

const extension = defineHardBreak()
const editor = createEditor({ extension })
// ---cut---
editor.commands.insertHardBreak()
```

## Keyboard Shortcuts

| Non-Apple     | Apple           | Description                                                  |
| ------------- | --------------- | ------------------------------------------------------------ |
| <Keyboard key="Ctrl+Enter" /> | <Keyboard key="Command+Enter" /> | Insert a hard break node at the current text cursor position |
| <Keyboard key="Shift+Enter" /> | <Keyboard key="Shift+Enter" />   | Insert a hard break node at the current text cursor position |
