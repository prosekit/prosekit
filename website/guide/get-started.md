# Get Started

## Installation

You can install ProseKit using `npm`, `yarn`, or `pnpm`. Choose the command that corresponds to your package manager:

::: code-group

```shell [npm]
npm install prosekit
```

```shell [yarn]
yarn add prosekit
```

```shell [pnpm]
pnpm add prosekit
```

:::
