The `italic` mark is used to represent text that is using an italicized type. It will be rendered as `<em>` element in HTML.

<Demo story="italic" />

## Commands

### `toggleItalic`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineItalic } from 'prosekit/extensions/italic'

const extension = defineItalic()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleItalic()
```

## Keyboard Shortcuts

| Non-Apple | Apple       | Description                            |
| --------- | ----------- | -------------------------------------- |
| <Keyboard key="Ctrl+I" /> | <Keyboard key="Command+I" /> | Toggle the current selection to italic |
