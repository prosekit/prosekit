The `strike` mark is used to represent text with a strikethrough. It will be rendered as `<s>` element in HTML.

<Demo story="strike" />

## Commands

### `toggleStrike`

```ts twoslash
import { createEditor } from 'prosekit/core'
import { defineStrike } from 'prosekit/extensions/strike'

const extension = defineStrike()
const editor = createEditor({ extension })
// ---cut---
editor.commands.toggleStrike()
```

## Keyboard Shortcuts

| Non-Apple                       | Apple                              | Description                            |
| ------------------------------- | ---------------------------------- | -------------------------------------- |
| <Keyboard key="Ctrl+Shift+X" /> | <Keyboard key="Command+Shift+X" /> | Toggle the current selection to strike |
| <Keyboard key="Ctrl+Shift+S" /> | <Keyboard key="Command+Shift+S" /> | Toggle the current selection to strike |
