Keyboard shortcuts are essential for creating a smooth editing experience. ProseKit makes it easy to add custom keyboard shortcuts to your editor using the `defineKeymap` function.

## Using defineKeymap

The `defineKeymap` function allows you to define custom keyboard shortcuts for your editor:

```ts twoslash
import {
  createEditor,
  defineKeymap,
} from 'prosekit/core'

const extension = defineKeymap({
  'Mod-b': (state, dispatch) => {
    console.log('Bold shortcut pressed')
    return true
  },
  'Shift-Enter': (state, dispatch) => {
    console.log('Shift-Enter pressed')
    return true
  },
})

const editor = createEditor({ extension })
```

## Key modifiers

ProseKit supports the following modifiers:

- `Mod`: Maps to `Command` on macOS and `Ctrl` on other platforms
- `Ctrl`: Control key
- `Alt`: Alt key (Option on macOS)
- `Shift`: Shift key
- `Meta`: Command key on macOS, Windows key on other platforms

<Aside type="tip">
Use `Mod` instead of `Ctrl` or `Meta` to create cross-platform shortcuts that work correctly on both macOS and other operating systems.
</Aside>
